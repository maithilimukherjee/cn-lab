# CN Lab

## Overview
This repository contains laboratory exercises and implementations for Computer Networks (CN) coursework. It covers fundamental networking concepts, protocols, and network programming through hands-on Java implementations.

## Objectives
- Understand core networking concepts and OSI model layers
- Implement and experiment with network protocols
- Develop socket-based client-server applications
- Explore data transmission, routing, and error handling
- Gain practical experience with network communication

## Content Areas
The lab typically covers:
- **Socket Programming**: TCP/UDP client-server implementations
- **Protocol Implementation**: Custom and standard networking protocols
- **Network Utilities**: Tools for packet analysis and network diagnostics
- **Error Handling**: Timeout management, connection handling, and recovery
- **Data Serialization**: Message formatting and transmission

## Technology Stack
- **Language**: Java
- **Tools**: Standard Java networking libraries (java.net, java.io)
- **Build System**: Maven/Gradle (optional)


## Getting Started
1. Clone the repository
2. Navigate to the project directory
3. Review the exercise documentation
4. Implement the required programs
5. Test with provided test cases

## Prerequisites
- Java JDK 8 or higher
- Basic understanding of networking concepts
- Familiarity with Java programming
